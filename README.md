# 🥗 NutriCalc – Budget-Based Nutrition Recommendation System

## 📌 Overview
NutriCalc is a full-stack web application that generates personalized 7-day meal plans based on a user’s physical profile, dietary goals, and budget constraints. The system is designed to make nutrition planning affordable and practical, especially for students and individuals with limited budgets.

---

## 🎯 Features
- Personalized 7-day meal plans
- Budget-based food recommendations
- BMR & TDEE calculation using Mifflin–St Jeor equation
- Rule-based recommendation engine
- Secure user authentication system
- 600+ food dataset integration
- REST API-based backend
- Dynamic database-driven meal generation

---

## 🧠 How It Works
1. User inputs personal data (age, weight, height, activity level)
2. System calculates calorie needs (BMR + TDEE)
3. Applies goal-based adjustment (loss/gain/maintain)
4. Selects meals based on:
   - Calorie targets
   - Protein requirements
   - Budget constraints
5. Generates a complete 7-day meal plan

---
